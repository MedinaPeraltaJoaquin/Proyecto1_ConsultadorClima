Para la correcta ejecucion y funcionalidad del programa se requiere instalar node.js y otras dependencias.
link directo para instalacion de node.js: https://nodejs.org/es/download.
una vez instalado node.js abra la terminal y ejecute los siguientes comandos de instalacion:
npm install --save-dev jasmine
Una vez aseguradas estas dependencias debera ejecutar en la terminal el comando para iniciar el programa:
npm start
Si desea correr las pruebas unitarias del programa pueden ejecutarse escribiendo en la terminal el comando:
npm test
Esperemos el servicio sea de su agrado!!
